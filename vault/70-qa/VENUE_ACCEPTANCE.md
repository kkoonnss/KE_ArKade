# Venue Acceptance Script & Performance Budget

## Venue Acceptance Script (Checklist)
1. [ ] Power-cycle the system.
2. [ ] Wait for automatic restore (launcher up, UI visible).
3. [ ] Perform optical check (alignment, brightness, focus).
4. [ ] Launch Cartridge 1: Lumen Maze. Ensure input works, then Quit.
5. [ ] Launch Cartridge 2: Neon Stack. Ensure input works, then Quit.
6. [ ] Launch Cartridge 3: (Third Title). Ensure input works, then Quit.
7. [ ] Trigger Panic Black (kill switch). Screen must go completely black immediately.
8. [ ] Restore from Panic Black. UI must recover within 5 seconds.

## Performance Budget

### Profile L (Laptop, Wall Projection)
- Target FPS: 60fps stable.
- Frame Time: < 16.6ms.
- Latency: < 50ms end-to-end (input to photon).
- Memory: < 2GB active.
- Scene switch time: < 10 seconds.
- Hub overhead: < 5% CPU while game runs.
